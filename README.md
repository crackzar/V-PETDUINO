# VPET-arduino
VPET  con arduino y display nokia 5110
 material necesario 
 
 *arduino uno "similares"
 *3 push button
 *1 speaker pasivo
 *display nokia 5110
 *5 resistencias de 10K
 *1 resistencia de 1 K
